# Example: Creating a Small Skill

Prompt:

```text
Use $virtual-company-workflow to create a small cross-platform skill.
Keep the process lightweight.

Goal:
Create a skill that helps agents run projects like a small company with roles for user representative, product manager, designer, architect, developer, QA, and reviewer.

Constraints:
- It should work in Codex, Claude Code, and generic LLM tools.
- Avoid complex automation.
- Produce a minimal SKILL.md and a few reference files.
```

Expected flow:

1. User representative checks whether the workflow solves a real coordination problem.
2. Product manager defines the smallest useful skill.
3. Architect proposes a portable folder structure.
4. Developer creates the files.
5. QA/reviewer checks that the skill is concise and platform-neutral.
