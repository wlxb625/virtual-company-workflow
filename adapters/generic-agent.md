# Generic Agent Adapter

For tools without a formal skill system:

1. Paste `skills/virtual-company-workflow/SKILL.md` into project instructions.
2. Paste selected reference files only when needed.
3. Ask the agent to use single-agent mode unless the tool supports subagents.

Recommended prompt:

```text
Use this virtual company workflow.
You are the project lead. Simulate the roles sequentially.
Do not let the roles free-chat. Summarize trade-offs and ask me for decisions.
```
